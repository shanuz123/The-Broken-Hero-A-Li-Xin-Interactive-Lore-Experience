# ⚔️ Li Xin: The Broken Hero — Interactive Lore Experience
## A cinematic, interactive web experience exploring the tragic duality of Li Xin (李信). Built as a tribute to the "Broken Hero" for the Honor of Kings Community Fan Art Event.

## 📖 The Vision
As a developer who fell in love with the depth of the Honor of Kings universe, I wanted to create more than just a static page. Li Xin’s story is defined by conflict—the struggle between his former glory as a Qin general and the abyssal power he embraced for redemption.

This project uses modern web development techniques to let the user physically feel that shift, sliding between the Radiant Form (🌞) and the Shadow Form (🌑).

## ✨ Key Features
- Duality Split-Slider: A custom-coded UI element that reveals Li Xin’s two forms as the user interacts.
- Parallax Storytelling: As you scroll, the lore unfolds with layered background movements inspired by Chinese aesthetic grandeur.
- Dynamic Art Swapping: Seamlessly transition between imperial golden visuals and the chaotic crimson/purple of the abyss.
- Responsive Design: Optimized for both desktop and mobile viewing.

## 🛠️ Built With
- HTML5 - Semantic structure for high-end storytelling.
- CSS3 - Custom animations, glassmorphism UI, and clip-path for the slider mechanics.
- JavaScript (Vanilla) - High-performance intersection observers and event listeners for interactive elements.

## 📜 Lore Chapters Included
- Vanguard (锋芒): The rise of the Qin prodigy.
- Fall (陨落): The catastrophic defeat that shattered a legend.
- Abyss (深渊): The forbidden bargain for power.
- Schism (分裂): The violent internal war of two souls.
- Eternal (永恒): The legacy of a hero who refuses to die.
